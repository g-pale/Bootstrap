# Проект: Природа России

## Описание проекта

Веб-сайт "Природа России" — это информационный ресурс, посвященный природным зонам и регионам России. Проект выполнен с использованием Bootstrap 5.3.3 и представляет собой одностраничное приложение с интерактивными элементами.

## Файл DZ_VD03.html

### Общая информация

- **Тип файла**: HTML5
- **Фреймворк**: Bootstrap 5.3.3
- **Язык интерфейса**: Русский
- **Кодировка**: UTF-8

### Структура и содержимое

#### 1. Заголовок страницы
- Главный заголовок "ПРИРОДА РОССИИ" с использованием Bootstrap классов:
  - `text-primary` — синий цвет текста
  - `fw-bold` — жирное начертание
  - `text-center` — центрирование
  - `py-4` — вертикальные отступы

#### 2. Навигационная панель (Navbar)
- Адаптивная навигация с поддержкой мобильных устройств
- **Элементы меню**:
  - **Контакты** — ссылка на футер страницы
  - **Описание** — переход к секции с карточками регионов
  - **Статьи** — переход к аккордеону с подробной информацией
  - **Ссылки на ресурсы** (выпадающее меню):
    - Фото — переход к карусели изображений
    - Википедия — внешняя ссылка на категорию Википедии
- **Поиск**: форма поиска с кнопкой в правой части навбара

#### 3. Карусель изображений (Carousel)
- Автоматически прокручивающаяся карусель с тремя изображениями:
  - Дальний Восток России (`DVostok.png`)
  - Сибирь (`Siberia.png`)
  - Средняя полоса России (`Middle.png`)
- Навигационные кнопки для ручного переключения слайдов
- Автоматическое воспроизведение (`data-bs-ride="carousel"`)

#### 4. Секция описания регионов (Cards)
- Три карточки с информацией о регионах России:
  1. **Природа средней полосы России**
  2. **Природа Сибири**
  3. **Природа дальнего востока**
- Каждая карточка содержит:
  - Изображение региона
  - Заголовок
  - Краткое описание
  - Кнопку-ссылку на соответствующую статью в Википедии
- Адаптивная сетка Bootstrap:
  - На мобильных устройствах: 1 колонка (`col-sm-12`)
  - На планшетах: 2 колонки (`col-md-6`)
  - На десктопах: 3 колонки (`col-lg-4`)

#### 5. Аккордеон со статьями
- Интерактивный аккордеон с тремя разделами:
  1. **Дальний Восток России** (открыт по умолчанию)
  2. **Средняя полоса России**
  3. **Сибирь**
- Каждый раздел содержит подробную информацию:
  - Общее описание региона
  - Рельеф и климат
  - Флора и фауна
  - Природные ресурсы
  - Природные достопримечательности
- Возможность разворачивания/сворачивания каждого раздела

#### 6. Футер
- Информационный блок с контактными данными:
  - Название организации: ООО "Природа России"
  - Телефон: +7 (495) 777-77-77
  - Email: priroda@mail.ru
  - Ссылка на социальную сеть ВКонтакте (оформлена как кнопка Bootstrap)
- Стилизация Bootstrap:
  - Светлый фон (`bg-light`)
  - Вертикальные отступы (`py-4`)
  - Центрирование текста (`text-center`)
  - Идентификатор `Contact` для якорной ссылки из навбара
  - Кнопка ВКонтакте использует классы `btn btn-outline-primary btn-sm` для соответствия общей стилистике страницы
- Ссылка на ВКонтакте открывается в новой вкладке (`target="_blank"`)

### Функциональность

1. **Адаптивный дизайн**
   - Полностью адаптивная верстка для всех типов устройств
   - Мобильное меню с кнопкой-гамбургером
   - Адаптивная сетка карточек

2. **Навигация**
   - Якорные ссылки для быстрого перехода к разделам страницы
   - Внешние ссылки (Википедия, ВКонтакте) открываются в новой вкладке
   - Активное состояние текущего пункта меню

3. **Интерактивные элементы**
   - Автоматическая карусель изображений
   - Интерактивный аккордеон с возможностью разворачивания разделов
   - Выпадающее меню в навигации
   - Форма поиска (UI элемент, функциональность не реализована)

4. **Визуальные эффекты**
   - Плавные переходы и анимации Bootstrap
   - Hover-эффекты на кнопках и ссылках
   - Визуальная обратная связь при взаимодействии

### Используемые технологии

- **HTML5** — разметка страницы
- **Bootstrap 5.3.3** — CSS-фреймворк для стилизации и компонентов
- **JavaScript** — Bootstrap JS для интерактивных компонентов (подключается через CDN)

### Зависимости

Все зависимости подключаются через CDN:
- Bootstrap CSS: `https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css`
- Bootstrap JS: `https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js`

### Изображения

Проект использует следующие изображения (должны находиться в той же директории):
- `DVostok.png` — изображение Дальнего Востока

- `Middle.png` — изображение средней полосы России

### Особенности реализации

- Семантическая HTML-разметка
- Использование стандартных компонентов Bootstrap
- Доступность (ARIA-атрибуты для навигации и форм)
- Оптимизация для различных размеров экранов
- Чистый и структурированный код

### Рекомендации по использованию

1. Убедитесь, что все изображения находятся в той же директории, что и HTML-файл
2. Для работы интерактивных элементов требуется подключение к интернету (загрузка Bootstrap через CDN)
3. Рекомендуется использовать современный браузер с поддержкой HTML5 и CSS3
